# Creating shortcuts in Microsoft Fabric Lakehouse

1. Go to your respective workspace, in this case **(Dev) DC Ops -- Ch** and click on the Lakehouse folder.

![Creating_Shortcuts_in_Fabric_Lakehouse_image1](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image1.png)

2. Within the folder select the Lakehouse in which you want to create your shortcut.

![Creating_Shortcuts_in_Fabric_Lakehouse_image2](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image2.png)

3. In Explorer under **Tables**, click the **⋯** (ellipsis) and choose **New table shortcut → Table shortcut**.

![Creating_Shortcuts_in_Fabric_Lakehouse_image3](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image3.png)

![Creating_Shortcuts_in_Fabric_Lakehouse_image4](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image4.png)

5. Choose **Microsoft OneLake**.

![Creating_Shortcuts_in_Fabric_Lakehouse_image5](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image5.png)

6. Select the **source Lakehouse** (often **LH_SOURCE_DATA**).

![Creating_Shortcuts_in_Fabric_Lakehouse_image6](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image6.png)

7. Pick the table(s) to shortcut (e.g., **sn_cmn_location**), then click **Next**.

![Creating_Shortcuts_in_Fabric_Lakehouse_image7](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image7.png)

8. Use the **pencil (edit)** icon to rename if needed. If a name already exists, Fabric silently appends **_1**.

![Creating_Shortcuts_in_Fabric_Lakehouse_image8](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image8.png)

9. Click **Create**.

![Creating_Shortcuts_in_Fabric_Lakehouse_image9](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image9.png)

10. Your shortcut is created.

![Creating_Shortcuts_in_Fabric_Lakehouse_image10](https://cdostorageacct.blob.core.windows.net/fabric-chatbot/fabric_images/Creating_Shortcuts_in_Fabric_Lakehouse_image10.png)
